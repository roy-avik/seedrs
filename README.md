* Database creation and running server

RUN gem install bundler
RUN bundle install
RUN rails db:setup
RUN rails s

RUN rspec

* DOCKER COMPOSE
From CMD or Bash

docker-compose build

docker-compose up

or

simply run the ./start.sh from BASH
